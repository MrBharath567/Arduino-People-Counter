# Arduino-Based People Counter & Automatic Light Control
My mini project is based on the home Automation using Arduino, it controls the room when the person enters and ,it also it consists of people counting ,that how many entered the living in it.

## Project Description
This project counts the number of people entering a room and automatically controls lighting based on occupancy.

## Components Used
- Arduino Uno
- LDR Sensor
- IR Sensor
- LCD Display
- LEDs

## Features
- Automatic light control
- Energy saving system
- Real-time people counting
